# Proyecto-Pymes
## 🧠 Sistema de Gestión para PYMES

API backend desarrollada para la gestión de una pequeña/mediana empresa, permitiendo administrar productos, categorías y usuarios con persistencia en base de datos.

### 🚀 Descripción

Este proyecto implementa un backend que permite realizar operaciones CRUD sobre distintas entidades del sistema, facilitando la organización y control de información dentro de una pyme.

Incluye autenticación de usuarios y manejo de base de datos mediante ORM.

### 🛠️ Tecnologías utilizadas
 - Node.js
 - Express
 - Sequelize
 - SQLite
 - JavaScript

### ⚙️ Funcionalidades principales
 - Gestión de artículos
 - Gestión de categorías
 - Gestión de usuarios
 - Autenticación básica
 - Persistencia de datos con Sequelize
 - API REST estructurada

### ▶️ Cómo ejecutar
npm install  
npm start

### 📂 Estructura
/models → modelos de base de datos  
index.js → punto de entrada  
auth.js → lógica de autenticación
